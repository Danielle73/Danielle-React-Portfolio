A fast, accessible, and modern personal portfolio showcasing projects, experience, and design work.
This site highlights frontend development skills with a focus on performance, accessibility, and clean UI.

### Overview

This project is a fully responsive portfolio built with Vite, React and Tailwind CSS, and deployed on Vercel.
It demonstrates modern frontend practices including component-based architecture, semantic HTML, accessible UI patterns, and lightweight styling.

### Features

- Responsive design optimized for mobile, tablet, and desktop
- Fast Vite-based development environment
- Clean, accessible interface
- Project sections with dedicated pages
- Dark mode toggle
- Smooth navigation and consistent layout
- Contact and social links
- Deployed on Vercel with continuous deployment

Every push to the main branch triggers an automatic redeploy.
